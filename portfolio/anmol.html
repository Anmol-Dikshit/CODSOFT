TIC TAC TOE

This project is a web-based Tic Tac Toe game built using the MERN stack. It allows two players to take turns playing the classic game on a shared browser interface. The project demonstrates the integration of React for the user interface, Node.js and Express for the server-side logic, and (optionally) MongoDB.
